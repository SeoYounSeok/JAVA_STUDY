### 프롤로그

```
* 왜 자바스크립트를 공부를 고집하다가, 자바를 갑자기 공부를 하는가?
  - 얼마전 OO 기업 필기 시험을 본 경험이 있었다. 
    합격/불합격의 여부가 중요한 것이 아니라, 자바 문제가 40문제를 풀었는데 기본이 안되어 있다는 느낌을 받았다.
    직렬화의 개념 부터, 생성자, 추상클래스, 예외처리 등 다양한 내용에 대한 기초 부족 등 자바에 대한 개념이 전혀 없다는 느낌을 받았다.
    프론트엔트를 고집하는 것도 중요하지만, 재직중인 업무에 있어서 지장을 주고 싶지 않았으며, 내 자신에 대해 돌아보는 좋은 기회가 되었다.
    단순 정의나 내용을 확인하는 것이 끝이 아니라 조금 더 딥하게 공부를 해보자는 목표를 세운다.
```

### 목차

```
* 면접 준비 내용
  - 신입 개발자들을 채용할 때, 신입을 위한 기본 내용을 여쭤보신다.
    해당 내용들을 신입 개발자의 관점과 면접관분들께서 바라보시는 관점을 둘다 파악하고자 한다.

* 실제 코드 작성
  - 경력직 면접을 진행하면서 가장 중요하다고 느낀 것은, 경험이 있냐 없냐라는 것이다.
    어떤 작업에 대해 이론적으로 다 파악하고 있더라도, 그 상황을 마주쳤을 때 어떻게 처리했는지, 어떤 식으로 확인했는 지 등 여러가지가 중요하다고 생각한다.
    간단한 작업 및 중요하다고 생각되는 부분은 실제 코드를 만져봐서 확인할 예정이다.
```
